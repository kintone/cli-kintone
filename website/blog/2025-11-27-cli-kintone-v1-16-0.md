## What's new?

### New command: plugin init (Experimental)

We added a new command to initialize a kintone plugin project.

:::warning
This command is still under development.  
Please see our [Stability Index](/community/stability-index/#experimental) to learn how we update experimental features.
:::

#### [`cli-kintone plugin init`](/guide/commands/plugin-init/)

Initialize a new kintone plugin project.

```shell
cli-kintone plugin init \
  --name ${project_name} \
  --template ${template_name}
```
