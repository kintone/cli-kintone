## What's new?

### New commands: customize commands (Experimental)

We added new commands to manage kintone app customizations. These commands provide equivalent functionality to [@kintone/customize-uploader](https://www.npmjs.com/package/@kintone/customize-uploader).

:::warning
These commands are still under development.
Please see our [Stability Index](/community/stability-index/#experimental) to learn how we update experimental features.
:::

#### [`cli-kintone customize init`](/guide/commands/customize-init/)

Initialize a customize manifest file.

```shell
cli-kintone customize init --output customize-manifest.json
```

#### [`cli-kintone customize export`](/guide/commands/customize-export/)

Export customize settings from a kintone app to a manifest file.

```shell
cli-kintone customize export \
  --app 123 \
  --output customize-manifest.json \
  --base-url https://example.cybozu.com \
  --username admin \
  --password password
```

#### [`cli-kintone customize apply`](/guide/commands/customize-apply/)

Apply customize settings from a manifest file to a kintone app.

```shell
cli-kintone customize apply \
  --input customize-manifest.json \
  --app 123 \
  --base-url https://example.cybozu.com \
  --username admin \
  --password password
```

### Migration guide from js-sdk

We added a [migration guide](/guide/migration/migration-from-js-sdk/) to help users migrate from [@kintone/customize-uploader](https://www.npmjs.com/package/@kintone/customize-uploader) and other js-sdk tools to cli-kintone.
