## What's new?

### Official release of plugin and customize commands

We are excited to announce that the **plugin commands** and **customize commands** have graduated from Experimental status and are now officially released.

These commands were introduced as experimental features in previous releases ([v1.15.0](/blog/cli-kintone-v1-15-0/), [v1.16.0](/blog/cli-kintone-v1-16-0/), [v1.17.0](/blog/cli-kintone-v1-17-0/)) and have been stabilized based on user feedback and testing.

#### Plugin commands

The plugin commands provide functionality equivalent to [@kintone/plugin-packer](https://www.npmjs.com/package/@kintone/plugin-packer) and [@kintone/plugin-uploader](https://www.npmjs.com/package/@kintone/plugin-uploader).

| Command                                           | Description                                              |
| ------------------------------------------------- | -------------------------------------------------------- |
| [`plugin init`](/guide/commands/plugin-init/)     | Initialize a new kintone plugin project                  |
| [`plugin keygen`](/guide/commands/plugin-keygen/) | Generate a private key for plugin packaging              |
| [`plugin pack`](/guide/commands/plugin-pack/)     | Package plugin source code into a distributable zip file |
| [`plugin upload`](/guide/commands/plugin-upload/) | Upload a plugin to kintone                               |
| [`plugin info`](/guide/commands/plugin-info/)     | Display plugin information                               |

#### Customize commands

The customize commands provide functionality equivalent to [@kintone/customize-uploader](https://www.npmjs.com/package/@kintone/customize-uploader).

| Command                                                 | Description                                                     |
| ------------------------------------------------------- | --------------------------------------------------------------- |
| [`customize init`](/guide/commands/customize-init/)     | Initialize a customize manifest file                            |
| [`customize export`](/guide/commands/customize-export/) | Export customize settings from a kintone app to a manifest file |
| [`customize apply`](/guide/commands/customize-apply/)   | Apply customize settings from a manifest file to a kintone app  |
