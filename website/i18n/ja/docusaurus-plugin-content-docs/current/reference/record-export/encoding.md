# 文字エンコーディング

## コマンドラインインターフェース

```shell
// 出力文字エンコーディングを指定
cli-kintone record export --app 8 --encoding sjis
```

| パラメーター名 | 短縮名 | 説明               |
| -------------- | ------ | ------------------ |
| --encoding     |        | 文字エンコーディング |

## 文字エンコーディング

`--encoding`オプションを使用すると、出力結果の文字エンコーディングを選択できます。以下のエンコーディングがサポートされています：

| キー   | エンコード |
| ------ | ---------- |
| `utf8` | UTF-8      |
| `sjis` | Shift-JIS  |

デフォルトは`utf8`です

## エラー

- サポートされていないエンコーディングが指定された
