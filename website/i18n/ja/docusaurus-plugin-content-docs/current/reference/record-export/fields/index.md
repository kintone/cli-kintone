# フィールド仕様

基本的に、出力は[APIレスポンス](https://kintone.dev/en/docs/kintone/overview/field-types/)の`value`プロパティと同じです。

## レコード情報

| フィールドタイプ | 出力             |
| ---------------- | ---------------- |
| レコード番号     | APIレスポンスと同じ |
| レコードID       | サポートされていません |
| リビジョン       | サポートされていません |
| 作成者           | APIレスポンスと同じ |
| 作成日時         | APIレスポンスと同じ |
| 更新者           | APIレスポンスと同じ |
| 更新日時         | APIレスポンスと同じ |

## 一般フィールド

| フィールドタイプ | 出力                                                                                       |
| ---------------- | ------------------------------------------------------------------------------------------ |
| 文字列（1行）    | APIレスポンスと同じ                                                                        |
| 文字列（複数行） | APIレスポンスと同じ                                                                        |
| リッチエディター | APIレスポンスと同じ                                                                        |
| 数値             | APIレスポンスと同じ                                                                        |
| 計算             | APIレスポンスと同じ                                                                        |
| チェックボックス | APIレスポンスと同じ                                                                        |
| ラジオボタン     | APIレスポンスと同じ                                                                        |
| 複数選択         | APIレスポンスと同じ                                                                        |
| ドロップダウン   | APIレスポンスと同じ                                                                        |
| ユーザー選択     | [ユーザー選択、組織選択、グループ選択](./user-group-org-selection.md)                     |
| 組織選択         | [ユーザー選択、組織選択、グループ選択](./user-group-org-selection.md)                     |
| グループ選択     | [ユーザー選択、組織選択、グループ選択](./user-group-org-selection.md)                     |
| 日付             | YYYY-MM-DD<br/>例：2022-04-01                                                              |

| リンク           | APIレスポンスと同じ                                                                        |
| 添付ファイル     | [添付ファイル](./attachment.md)                                                            |
| ルックアップ     | サポートされていません                                                                     |
| テーブル         | [テーブル](./table.md)                                                                     |

| カテゴリー       | サポートされていません                                                                     |

| 作業者           | サポートされていません                                                                     |
| ラベル           | サポートされていません                                                                     |
| スペース         | サポートされていません                                                                     |
| 罫線             | サポートされていません                                                                     |
| グループ         | サポートされていません                                                                     |
