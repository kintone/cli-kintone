# 区切り文字

CSVからレコードをインポートする際、改行と区切り文字は以下の通りです。

| タイプ               | 値           |
| -------------------- | ------------ |
| レコード区切り文字   | 自動検出 \*1 |
| コンテンツ内の改行   | LF (`\n`)    |
| フィールド区切り文字 | カンマ (`,`) |

\*1: CRLF、LF、CRの中から適切なものを自動的に選択します。

複数のタイプの改行が使用されている場合、エラーが発生します。

自動検出には、csv-parseライブラリの機能を使用しています。

https://csv.js.org/parse/options/record_delimiter/
