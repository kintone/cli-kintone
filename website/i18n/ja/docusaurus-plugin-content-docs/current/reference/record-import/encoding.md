# 文字エンコーディング

## コマンドラインインターフェース

```shell
// 文字エンコーディングを指定
cli-kintone record import --app 8 --file-path records.csv --encoding sjis
```

| パラメーター名 | 短縮名 | 説明                             |
| -------------- | ------ | -------------------------------- |
| `--encoding`   |        | 入力ファイルの文字セットエンコーディング |

## 文字エンコーディングの指定

`--encoding`オプションを使用すると、CSVファイルをインポートする際の文字エンコーディングを指定できます。以下のエンコーディングが利用可能です：

| キー   | エンコード |
| ------ | ---------- |
| `utf8` | UTF-8      |
| `sjis` | Shift-JIS  |

デフォルトは`utf8`です

## エラー

- サポートされていないエンコーディングが指定された
- [エンコーディングの不一致](../errors/mismatch-encoding.md)
