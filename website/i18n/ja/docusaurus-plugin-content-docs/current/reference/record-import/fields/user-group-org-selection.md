# ユーザー選択、組織選択、グループ選択

## CSVファイル形式

### CSVファイルの例

```csv
"Record_number","User_selection","Group_selection","Department_selection"
"2","user01
user02
user03","Dev team1
Dev team2","Develepment dept.
Sales dept."
"1","user02","Dev team1","Sales dept."
```

### ユーザー選択フィールド

- 列名は、ユーザー選択フィールドのフィールドコードと等しい名前である必要があります。
- 列には、ユーザーのログイン名が含まれている必要があります。
- 複数のユーザー名は改行文字（LF）で区切る必要があります

### グループ選択フィールド

- 列名は、グループ選択フィールドのフィールドコードと等しい名前である必要があります。
- 列にはグループコードが含まれている必要があります。
- 複数のグループは改行文字（LF）で区切る必要があります

### 組織選択フィールド

- 列名は、組織選択フィールドのフィールドコードと等しい名前である必要があります。
- 列には組織コードが含まれている必要があります。
- 複数の組織は改行文字（LF）で区切る必要があります
